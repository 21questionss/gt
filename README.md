# Football Manager (simples)

Pequeno gerenciador de campeonato de futebol em Python — CLI.  
Permite:
- Adicionar/ler times;
- Gerar tabela de classificação;
- Agendar partidas;
- Simular partidas (aleatórias) ou inserir placares manualmente;
- Salvar e carregar dados em `data.json`.

Requisitos
- Python 3.8+

Como usar
1. Coloque `football_manager.py` no repositório.
2. Execute:
   ```bash
   python football_manager.py
